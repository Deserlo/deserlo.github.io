Portfolio

Web Scraping Project

Using Python, Selenium, and the BeautifulSoup library, I developed web scrapers to pick music event listings from various websites which saves the data to flat files. It uses Selenium, a testing automation framework, to extract the data from dynamic pages. I added complex logic with the utilization of the Youtube API which is used to search for a music video of the event artist. I designed and hand-coded the front-end of the website using HTML, CSS, and vanilla Javascript maintaining the responsive design for mobile, tablet, and desktop screens. My webserver executes a Python script daily to update the listings which writes to static files but new listings need to be added through manual execution of the scraping scripts.

Currently redesigning and implementing a new UI while also migrating to a Flask app. New web app currently deployed on Heroku for staging while in development.

NodeJS Web Application

My senior project team and I built a web application for a small, local martial arts studio in Rocklin, Ca. Our tech stack included Node.js, MongoDB, Express.js, and jQuery. The business had an existing website but was in need of a better UI design and some additional featues such as registered accounts, taking student attendance, and tracking progress.

As a main contributor to this project, major responsibilties were in the backend and included setting up an automated build using Heroku to deploy once any commmits were pushed to our master branch, designing the database schema, and developing the application structure to achieve high-cohesiveness and loose coupling between the modules.

Also, I effectively used jQuery to render the data on the front-end from our back-end.
